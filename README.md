# multi-robot-fleet-manager
📌 Key FeaturesMulti-Robot Path Planning: Simulation of up to 10 differential drive robots in grid-based environments.
Algorithms: Comparative analysis of $A^*$, Dijkstra, and Breadth-First Search (BFS). 
Conflict & Deadlock Management: Advanced strategies like Wait-For Graph and Reactive Re-planning to solve bottlenecks in narrow corridors.
Kinematic Analysis: Real-time derivation of linear/angular velocities and wheel speeds ($\omega_L, \omega_R$) based on the differential drive model. 
⚙️ Technologies & ToolsMATLAB App Designer: For creating the interactive Fleet Manager UI. Robotics System Toolbox: Occupancy grid mapping and distance calculations.
